# ZO Syndicate RP - Gang Logos, Patches & Flags

Professional gang branding for all 10 ZO Syndicate RP gangs.

## Overview

Each gang has 3 designs (SVG + PNG):
1. **Logo** (1024x1024) - Primary emblem for Discord, website headers, in-game textures
2. **Patch** (512x512) - Circular design for Discord emojis, profile pictures, jacket patches
3. **Flag** (1500x1000, 3:2 ratio) - Rectangular banner for territories, Discord banners, backgrounds

**Total Files**: 60 (30 SVG + 30 PNG)

## Usage

### In FiveM
- Logo PNG (1024x1024) can be used as gang territory markers, Discord server icons
- Patch PNG (512x512) perfect for in-game jacket patches, vehicle decals
- Flag PNG (1500x1000) ideal for gang hideout banners, territory flags

### On Website
- SVG files preferred for web (scalable, smaller file size)
- PNG fallback for older browsers or specific texture requirements

### In Discord
- Patch PNG (512x512) works as server emoji (resize to 128x128 for emoji)
- Logo PNG for gang role icons
- Flag PNG for channel banners

## Design Principles

- **Cultural Respect**: All designs honor real-world cultures while maintaining fictional gang identity
- **High Contrast**: Designs visible at all scales and on various backgrounds
- **Scalable**: SVG source files maintain quality at any resolution
- **FiveM Compatible**: PNG sizes optimized for GTA V texture streaming
- **Professional Aesthetic**: Intimidating yet polished gang branding
